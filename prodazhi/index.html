Продажи ЛАЙТ
Заказы (A:AN) + расчёт ЗП по “Дата сдачи заказа” (AN) + вкладка ЛИД • всё сохраняется в браузере
Готово
Поиск
Метрики считаются автоматически при вводе. ЗП: синий — к выплате, зелёный — выплачено.
Подсказка: чтобы “убрать” старые — ставь Скрыть. Можно включить “Показывать скрытые”.
ЗП
Начисление после заполнения “Дата сдачи заказа” (AN)
Итого: 0 ₽
Разбивка
Суммы берутся из столбцов ЗП в строке заказа
    Выплачено Дата сдачи Заказ Сотрудник Роль Сумма, ₽
    Логика периода: дата 1–15 → “1–15”, 16–конец → “16–конец”. Пример: 13.02.2026 → период 01–15.02.2026.
    Менеджер вручную заполняет только этапы Зая, Диз и КП. Этапы Счёт и Заказ считаются автоматически из вкладки “Заказы”.

    Справочники (выпадающие списки)
    Добавляй / удаляй значения — они сразу появятся в таблице заказов.
    Внимание
    Удаление значения из справочника не удаляет его из уже заполненных заказов, но оно исчезнет из выпадающего списка для будущего выбора.
